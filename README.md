# Exams Helper

A React app for practicing multiple-choice exams. It loads exam questions from JSON files, randomizes answer options, and gives immediate feedback on each answer along with a running score. New exams can be added by dropping a JSON file in the data folder and registering it in the app.

Built with React 19, TypeScript, Vite, and Tailwind CSS.

## Getting Started

```bash
pnpm install
pnpm dev
```
